# BFSTool
EXPERIMENTAL AND UNDER DEVELOPMENT A tool to format, read and write the Burstcoin FileSystem under Windows. 

BFS Tools for Windows
- Partition a drive to BFS
- Add Plotfiles
- Delete Plotfiles
- Merge PlotFiles
- Copy / Move Plotfiles
